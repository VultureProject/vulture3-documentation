Network settings

Overview

From this menu you can manage settings relative to the listeners on which your application should be available. You cannot modify or remove a listener while it is running.

Network settings

Click on "Add an entry" to add a new listener to your application.

  • IP Address: Select the listener on which you want Vulture to listen for incoming connection.
  • TCP port: The TCP port to listen onto.
  • SSL Profile: If you want TLS to be enable on the listener, select your TLS Profile here.
